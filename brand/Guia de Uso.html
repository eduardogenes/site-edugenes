Quando usar cada variação

00 · A ideia

genes → código genético → código-fonte

O símbolo é uma escada de pares de bases (A·T em café, C·G em verde) — o sobrenome Genes carregando a profissão. O underscore _ no fim do wordmark é o cursor do terminal: código vivo, sempre em construção.

Regra de ouro: nunca redesenhe nenhuma peça. Use sempre os arquivos desta pasta.

01 · Lockups

Três formações, uma hierarquia clara

Horizontal — principal O padrão. Header do site, topo de documentos, apresentações, assinatura de e-mail. Em dúvida, use este. png/lockup-horizontal-tinta.png · -papel.png
Empilhado — apoio Espaços quadrados ou verticais: capas, cards de compartilhamento, stories, etiquetas. png/lockup-empilhado-tinta.png · -papel.png
Wordmark sozinho — apoio Faixas horizontais baixas: rodapés, créditos, barras de navegação compactas. png/wordmark-tinta.png · -papel.png

Versões tinta = para fundos claros · versões papel = para fundos escuros. Fundo transparente em todas.

02 · Símbolo

Duotone é a assinatura; mono é a versátil

Duotone Sempre que houver cor disponível: digital, slides, impressos coloridos. É a versão que constrói memória da marca. svg/simbolo-duotone.svg · -para-fundo-escuro.svg
Monocromático Contextos de 1 cor: carimbo, bordado, gravação a laser, marca d'água, impressão P&B. svg/simbolo-mono-tinta.svg · -papel.svg

Pode

  • Usar o símbolo sozinho em contextos próprios (site, materiais seus)
  • Escalar à vontade — é SVG
  • Usar sobre café ou verde, na versão mono

Não pode

  • Mudar as cores das barras ou a ordem A·G·G·A
  • Rotacionar, espelhar ou animar o símbolo
  • Usar abaixo de 20px — aí é caso pro favicon

03 · Favicon & Avatar

Dois cortes oficiais, cada um no seu habitat

Favicon · só barras Site e abas de navegador, PWA, atalhos. É o corte do símbolo pra tamanhos minúsculos: sem os pontos, que viram poeira abaixo de 32px. png/favicon-16/32/48/64/180/192/512.png · svg/favicon-*.svg
Avatar · eg▍ cursor-bloco Perfis: GitHub, LinkedIn, redes, foto de e-mail. As iniciais são pessoais — é você, não o site. O cursor-bloco ecoa as pílulas do símbolo. png/avatar-eg-tinta.png · -claro.png · -cafe.png

Os avatares têm fundo cheio (quadrado) — as plataformas recortam o círculo sozinhas. Tamanhos de favicon: 16/32 (aba), 48/64 (atalhos), 180 (Apple touch), 192/512 (PWA/Android).

05 · O cursor

Quando pisca, quando para

R1

Pisca só em telas vivas — site, apps, apresentações em HTML. Ciclo de ~1,1s, e sempre respeitando prefers-reduced-motion.

R2

Estático com o cursor à vista em tudo que não anima: print, PDF, avatar, favicon, assinatura de e-mail. O cursor nunca some — só para de piscar.

R3

O símbolo (escada) nunca pisca. O gesto do cursor pertence ao wordmark e ao monograma, apenas.

07 · Implementação no site

Entregue o README.md ao Claude Code

O arquivo README.md desta pasta é o manual de implementação: diz onde cada arquivo vai no código (public/, tags do <head>, manifest, tokens CSS, o cursor piscando em CSS com prefers-reduced-motion) e as 7 regras obrigatórias.

Pra garantir que o Claude Code siga sempre: o README traz na seção 9 um bloco pronto pra colar no CLAUDE.md do seu repositório — assim toda conversa futura no seu código já nasce sabendo as regras da marca.
